Pandobot is a [Twitch](https://en.wikipedia.org/wiki/Twitch_(service)) streambot that can monitor stream events such as channel point redeems, follows/subscriptions, chat messages, and more in order to play custom corresponding stream events.

Since pandobot runs locally on my computer during streams on one end and interfaces with Twitch API on the other, it bridges the gap between the stream interface and my local computer allowing for fun/interesting viewer interactions like manipulating my controls, screen, or game if compatible.

I know there are existing applications for features like this like [Streamerbot](https://streamer.bot/) and [Mix it up](https://mixitup.bot/) but I wanted to try to make my own and have full control over everything it can do.
